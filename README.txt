== Description ==

**Instantly Admin / Owner gets a notification on WhatsApp for each Contact Form 7 submission!**
Immediately get notified on WhatsApp when someone tries to reach out to you by filling out the contact form on your website. You can then immediately call them or take necessary action to improve your business. 

This simple addon widget adds functionality to the amazing Contact Form 7 (CF7) plugin so that every contact submission is received by you on your WhatsApp.

The plugin comes with a built-in anti-spam functionality also so that you are not bombarded with unnecessary spammy messages. Additionally, you don't have to install any other plugin, or software, or even purchase anything to use this plugin.

You will need Contact Form 7 version 3.0+ at a minimum. It is recommended to use version 5.3+ of CF7, for better spam logging. For the best results, we suggest always using the latest versions of WordPress and CF7.


== PRIVACY ==
  This plugin does not track users, or store any user data, nor does it use any cookies. This is just an addon plugin and requires Contact Form 7. Please review Contact Form 7’s privacy policies for more information.


== Installation ==

= From Dashboard ( WordPress admin ) =
* plugins -> Add New
* search for 'Message Notification for Contact Form 7'
* click on Install Now and then Active.

= using FTP or similar =
* Unzip the "Whatso" file and
* Upload "Whatso" folder to the "/wp-content/plugins/" directory.
* Activate the plugin through the "Plugins" menu in WordPress.

== Frequently Asked Questions ==

= Is this plugin free of cost? =
Yes, It is absolutely free of cost. There is no pro version and no need for you to upgrade to any other plan.

= Are there any limits on the number of contact form 7 submission? =
There is no limit on the number of contact form 7 submissions and you will receive each notification.

= I am not receiving the message. What could be the reason? =
If you are not receiving the message, please check if the plugin is enabled. If the plugin is enabled, then do check if the message contains a URL because this plugin does not support URLs to avoid any kind of spam. 

= Do you provide any support? =
Support is free of cost and you can reach out to us via hi@whatso.net

= Is this plugin directly associated with the original Contact Form 7? =
This plugin is not directly affiliated with the Contact Form 7 founder and is just a simple add-on to enhance CF7 functionality.
